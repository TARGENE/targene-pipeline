Project Organization

The TarGene project is organized around the targene-pipeline repository which contains the Nextflow pipeline which will be the entry-point for most users. However, this repository does not contain the executables that are used by the Nextflow processes. Those executables originate from complementary repositories:

  • TargetedEstimation.jl
  • TargeneCore.jl
  • UKBMain.jl
  • NegativeControl.jl
